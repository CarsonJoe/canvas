## Konva 2D Canvas Engine

- **Stage** → Layer → Shape hierarchy (react-konva)
- Shapes: `Rect`, `Ellipse`, `Line`, `Arrow`, `Text`, `Image`
- Custom `StrokeLine` with per-point stylus pressure data
- `Transformer` node for multi-select, resize, and rotation
- `useImage` hook for loading image frames
- Hit detection & drag-and-drop built-in to Konva
- Viewport: pan via `stageX/Y`, zoom via `stageScale`
