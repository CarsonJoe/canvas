## CLI Commands — `canvas`

```bash
canvas serve    # stdio MCP + local web app (default)
canvas http     # HTTP MCP + local web app
canvas setup    # interactive MCP client setup guide
canvas doctor   # diagnostics & health check
canvas version  # version info
```

Install: `npx @cogniboom/canvas setup`

**Supported MCP clients (10+)**

Claude Desktop · Cursor · Windsurf · VS Code
Cline · Zed · Continue · Goose · Kiro · Witsy
